👨‍🏫 專業導遊群

我們嚴選具備豐富在地知識與熱忱的導遊，為您的旅程保駕護航：

花花 (Hana)
日本通 / 購物達人

精通日語，對東京的大小巷弄如數家珍，最擅長幫旅客規劃最順路的購物與甜點行程。

小智 (Ash)
韓國深度遊 / 街頭美食家

在首爾留學四年，能帶你避開觀光客陷阱，品嚐只有當地人才知道的隱藏版韓式料理。

霏霏 (Fifi)
四川專家 / 攝影好手

具有專業攝影師證照，特別熟悉畢棚溝與九寨溝的最佳拍照角度，保證讓您的社群貼文讚數爆表。

仁人 (Ren)
泰國玩家 / 歷史講解

熱愛東南亞文化，講述大皇宮與古蹟故事生動有趣，是團員公認最幽默的導遊。
